# Auto Form

Pass a simple configuration object to your form, and let it do the rest. Piece of cake.

## Functionality

- Automatic error, success, loading messages.
- Callback function.
- Simple validation

## Config

```
{
    {
        label: '',
        name: '',
        type: '',
        required: false,
        success: '',
        error: ''
    }
}
```
